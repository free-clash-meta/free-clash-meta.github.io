详细安装指南

第一步：获取官方安装包

重要提示：务必通过Surfboard官网下载安装包，避免第三方渠道可能带来的安全风险。

  1. 打开浏览器访问官网
  2. 点击"Downloads"进入下载页面
  3. 根据您的操作系统选择对应版本：
    • Windows用户：选择.exe格式安装包（支持Win10及以上）
    • Mac用户：下载.dmg镜像文件（兼容macOS 10.15+）
    • Linux用户：获取.deb或.rpm包（适配主流发行版）

专业建议：下载前建议校验文件SHA-256值，确保安装包完整性。

第二步：安装过程详解

Windows系统安装

  1. 双击下载的.exe文件
  2. 在用户账户控制提示中选择"是"
  3. 按照向导完成以下步骤：
    • 选择安装语言（推荐英文或简体中文）
    • 阅读并同意许可协议
    • 指定安装路径（默认C:\Program Files\Surfboard）
    • 勾选创建桌面快捷方式
  4. 点击"安装"按钮，等待进度条完成
  5. 取消勾选"立即运行"选项（建议先完成配置再启动）

macOS系统安装

  1. 打开下载的.dmg磁盘映像
  2. 将Surfboard图标拖拽至Applications文件夹
  3. 在启动台中找到并打开应用
  4. 首次运行时需在系统偏好设置中授予网络权限

常见问题：若出现"无法验证开发者"提示，需前往"安全性与隐私"手动允许运行。

第三步：初始配置

成功安装后，首次启动将进入配置向导：

  1. 服务器设置

    • 手动输入：如果您有自己的服务器，填写地址、端口、加密方式等信息
    • 订阅链接：支持一键导入标准格式的订阅URL

  2. 代理模式选择

    • 全局模式：所有流量都通过代理
    • 智能分流：自动识别国内外网站（推荐日常使用）
    • 自定义规则：高级用户可编写自己的路由规则

  3. 高级选项

    • 设置本地监听端口（默认1080）
    • 开启UDP转发（适合游戏和视频通话）
    • 配置DNS防污染（建议使用Cloudflare或Google DNS）

1. 确认本地防火墙未阻止Surfboard
2. 重置IE代理设置（即使不使用IE也可能影响）

FAQ

WinXray 的内核能否替换？
可以。WinXray 的内核文件独立存在于程序目录中，用户可替换为最新的 v2ray-core 或 xray-core 文件以获取更好的性能与协议支持，但需保持版本兼容。
Nekoray 的订阅更新机制如何运作？
Nekoray 可以通过远程订阅 URL 自动下载最新节点和规则文件。用户可以设置更新频率，程序会定时检测并替换旧节点，保持配置最新状态。
Matsuri 节点延迟排序如何优化流量？
Matsuri 会定期测速节点延迟，并结合策略组自动选择低延迟节点。无需手动切换，确保网页、视频和游戏流量通过最优节点，提高访问速度和网络稳定性。
Mihomo 支持节点分类管理吗？
支持。用户可将节点按地区、用途或延迟分类，结合策略组和规则进行分流。这样可以快速选择最佳节点，提高访问效率并便于维护大量节点。
VMess 连接不稳定的常见原因有哪些？
VMess 连接不稳定可能由节点延迟过高、服务器防火墙、TLS 配置错误或 ISP 丢包引起。建议测试不同传输协议（如 WS、gRPC）或切换节点，检查日志确认具体错误信息。
V2RayN 订阅更新后节点重复如何清理？
V2RayN 在导入订阅时有时会产生重复节点，可以点击右键菜单选择“删除重复项”。若仍存在旧节点缓存，可清空配置文件夹内的 history.json 并重新导入订阅，保证节点列表整洁。
VMess 配置 TLS+WebSocket 有哪些性能优势？
TLS+WebSocket 可伪装 HTTPS 流量，增强抗封锁能力，同时降低被封检测风险。结合策略组和节点延迟检测，可保证访问速度和连接稳定性，适合视频和游戏场景。
Potatso Lite 能否兼容最新的 iOS 系统？
Potatso Lite 更新频率较低，在新版本 iOS 上可能出现兼容性问题。若遇到无法连接或闪退情况，建议升级至其他活跃维护的客户端如 Shadowrocket 或 Quantumult X。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
Clash 的 Profile URL 是否可以同时添加多个？
可以。Clash 支持导入多个 Profile URL，用户可在“配置”面板切换不同订阅源。多个配置之间互不干扰，方便区分不同地区或用途的线路。
